# BulkMail Project

A simple **bulk email sending application** built with **Node.js**, **Express**, **MongoDB**, and **React**.  
This project allows users to send emails programmatically through a backend API and manage email sending efficiently.

---

## Features

- Send emails using **Nodemailer**.
- Connect to **MongoDB** for storing configurations or logs.
- Simple **React frontend** for entering email messages and triggering send.
- CORS enabled for smooth frontend-backend communication.
- Easily configurable using **environment variables**.

---

## Tech Stack

- **Backend:** Node.js, Express, Nodemailer  
- **Database:** MongoDB  
- **Frontend:** React.js (Vite or Create React App)  
- **Others:** dotenv, cors, axios
